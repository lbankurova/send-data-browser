## Decisions made this session

- **LOO is a continuous multiplier, not a binary classifier.** Simulation proved 50% FPR under null at all N. The 0.8 threshold is display-only.
- **No asymmetric scoring penalty.** Simulation proved treated/control sides are symmetric (delta <0.01).
- **Bidirectional LOO adds value only at N>=10.** Masking effect at N=3-5 means LOO under-detects control outliers.
- **HCD percentile context is essential for nonrodent** but deferred for real technical dependencies.
- **Architect gate simplification:** Removed `loo_control_fragile` from `AdverseEffectSummaryRow` and `EndpointSummary`. Per-sex path only (4 propagation points instead of 7).
- **gLower anti-conservatism is not a defect.** The 0.3 gate passes 12-16% of null draws, and no consumer uses it alone.
