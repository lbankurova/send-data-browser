## Decisions made this session

- **Verify pain point before refactoring** — audit findings are hypotheses, not instructions. Read the code, verify the problem is real.
- **Namespace split: lattice: vs ops:** — user-facing workflows vs agent-autonomous tools. Keeps autocomplete clean.
- **Design step is mandatory before frontend implementation** — /lattice:design sits between synthesize and implement.
- **Visual Context Window model** — screen budget is like context window. Proximity hierarchy: eye movement (free) > filter toggle > tab > scroll > view switch > modal (expensive). Every scroll is a context eviction.
- **Charts and tables are orthogonal** — charts show patterns, tables show exact values. Need both, design decides which gets prime space.
- **Don't build custom component library** — app migrates to Datagrok. Invest in transferable decisions, not throw-away components. Some custom SVG may stay.
- **Chrome DevTools MCP for design iteration, Playwright for regression** — two tools for two jobs.
- **insights_engine.py stays** — cross-study comparison is coming, single-study views first.
