# Design Decision Log

> **Purpose:** Running record of design decisions with DG platform context. Every decision that has a Datagrok implication is logged here by the Review Agent after consultation with the DG Developer.
> **Who writes:** Review Agent (as part of the "closer" protocol)
> **Who reads:** DG migration team, all agents needing historical context
> **Format:** Newest entries at top.

---

## Template

```
### DD-NNN: [Short title]
- **Date:** YYYY-MM-DD
- **Task:** What was being implemented
- **Options considered:**
  1. [Option A] — [brief description]
  2. [Option B] — [brief description]
- **DG recommendation:** [What the DG Developer prescribed and why]
- **DG classification:** Native / Configure / Customize / Build
- **User decision:** [What the user chose, or "Accepted recommendation"]
- **Rationale:** [Why this decision was made]
- **Files affected:** [list]
- **Porting impact:** [What the DG migration team needs to know]
```
