Stage 3 — Dose-Response Pattern Classification
classify_dose_response()
Equivalence-Band Method Step-by-step dose comparison
against noise tolerance band band = frac × pooled_SD
step: |Δ| ≤ band → flat
        Δ > 0 → up
        Δ < 0 → down CLASS-02, STAT-11 | classification.py
Equivalence Fractions
CV-Tiered Thresholds
  • Tier 1 (CV<10%): 0.5 SD
    BW, RBC, protein, brain, heart
  • Tier 2 (CV 10-20%): 0.5 SD
    liver, kidney, ALT, AST, glucose
  • Tier 3 (CV>20%): 0.75 SD
    WBC, EOS, TRIG, spleen, thymus
Incidence: 1.5×√[p(1-p)/n]
floor = 2 percentage points
Output
Pattern Enum
  • monotonic_increase — all steps up
  • monotonic_decrease — all steps down
  • threshold_increase — flat then up
  • threshold_decrease — flat then down
  • non_monotonic — mixed
  • flat — all within band
  • insufficient_data — <3 groups
+ onset dose (threshold only)
+ confidence (HIGH/MOD/LOW)

classify_severity()
Incidence (direction-aware) Direction = up:

determine_treatment_related()
Boolean Gate (any sufficient)
  • Gate 1: p<0.05 AND trend_p<0.05
  • Gate 2: severity=adverse AND
    pattern=monotonic_*
  • Gate 3: p<0.01 (very significant)
→ treatment_related: bool CLASS-05, BFIELD-03 | classification.py
OM Two-Gate Assessment
_assess_om_two_gate() Gate 1 (Stat): p_adj < 0.05
Gate 2 (Mag): |pct_change| ≥ organ_threshold
  • Both gates + outside HCD → adverse
  • Both gates + within HCD → equivocal
  • Stat only + outside HCD → equivocal
  • Stat only + within HCD → non-adverse
  • Neither gate → not treatment-related
ceiling=5%, floor=10% (organ-specific) METH-33, BFIELD-74 | classification.py

Stage 8 — Signal Scoring & Confidence
Signal Score (0.0 - 1.0)
Continuous Domains score = 0.35×min(-log10(p)/4, 1)
      + 0.20×min(-log10(trend_p)/4, 1)
      + 0.25×min(|g|/2, 1)
      + 0.20×pattern_score Pattern scores: monotonic=1.0,
threshold=0.7, non_monotonic=0.3, flat=0 BFIELD-01 | view_dataframes.py
Signal Score (Incidence)
Incidence Domains score = 0.45×min(-log10(p)/4, 1)
      + 0.30×min(-log10(trend_p)/4, 1)
      + 0.25×pattern_score MI bonus: +0.10×min((sev-1)/4, 1) Capped at 1.0
GRADE-Style Confidence
6-Dimension Scoring
  • D1 (Stats): p<0.01+trend → +1
  • D2 (Pattern): monotonic=+1, non_mono=-1
  • D3 (Corrob): yes=+1, no=-1
  • D4 (HCD): outside=+1, within=-1
  • D5 (Sex): concordant+TR=+1, discord=-1
  • D6 (Band): zone check for Tier 2
≥2 → HIGH, 0-1 → MODERATE, ≤-1 → LOW METH-37, FIELD-63 | confidence.py
