## Method

For each (N, g, contamination) condition:

1. Draw control ~ N(0, 1), treated ~ N(g, 1)
2. Compute Hedges' g (small-sample corrected) and gLower (80% CI lower bound via non-central t)
3. Skip if gLower <= 0 (no detectable signal)
4. **Treated-side LOO:** remove each treated animal, recompute, record min(LOO-gLower)/gLower_full
5. **Control-side LOO:** remove each control animal, recompute, record min(LOO-gLower)/gLower_full
6. **Bidirectional:** min(treated-side, control-side)

Contamination conditions (g=1.0 only):
- **Control outlier:** One control from N(3, 1)
- **Treated outlier:** One treated from N(g+3, 1)
